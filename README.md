# the-pong-game

A minimal Python remake of the classic Pong game using the `turtle` module.

## Usage

Run the game with:

```bash
python main.py
```

Use W/S and Up/Down keys to move the paddles.

## Files

- `main.py` – game setup and loop
- `ball.py` – ball behavior
- `paddle.py` – paddle class and controls
- `scoreboard.py` – score display

That's it! Enjoy playing. 🎮
